# British Airways Reviews Dashboard

**Project Title**

British Airways Reviews

**Description**

The purpose of this project is to analyze passenger feedback and service ratings for British Airways to identify key pain points in the customer journey. By visualizing metrics across different seat types, aircraft, and travel demographics, the dashboard provides actionable insights into cabin service, seat comfort, and overall value for money.
